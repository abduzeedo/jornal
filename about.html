About

Remember when the web was simple? When you could view source on any page and actually understand what was happening? When sites loaded instantly because they were just HTML files?

Jornal is a return to that. No massive frameworks. No databases. No build steps that take forever. No dependencies to update. No security patches to worry about. Just HTML files served fast.

How It Works

Every page you see is a single HTML file. When you click a link, you get another HTML file. That's it. No server-side rendering, no JavaScript frameworks, no API calls. The result? Pages load instantly.

The styling uses Tailwind CSS through a CDNâ€”no build process required. The only JavaScript handles smooth page transitions. Everything else is just good old HTML and CSS.

AI Does the Heavy Lifting

Here's where it gets interesting. This entire site was built through conversation using Claude Code. Want a new blog post? Just describe it. Want to change the design? Just say how. Want to add a new page? Just ask.

Claude Code handles all the HTML, the file organization, updating the listings, and publishing to GitHub. You focus on what you want to say. The AI handles how it gets built.

Why This Matters

Somewhere along the way, we made websites complicated. We convinced ourselves we needed React for a blog. We added databases for sites that could be static files. We created build pipelines for projects that didn't need them.

Jornal is proof you don't need any of that. A blog can just be HTML files. And with AI, creating those files is as simple as having a conversation.

It's fast. It's simple. It's fun. Just like the web used to be.
